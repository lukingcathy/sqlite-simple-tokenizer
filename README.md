# sqlite-plugin-demo

> [!WARNING]
> 这个库正在开发中，还没有到可以使用的地步

> 这是一个使用 `rusqlite` 构建 SQLite fts5 插件的项目，其主要功能是为 SQLite 提供中文和拼音分词和检索。这个项目可以作为 Rust 的 crate 使用，也可以将其编译成动态库，在 SQLite 中加载和使用。
